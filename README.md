# Ethical Hacking of Windows 7 OS Using Linux – A VirtualBox-Based Approach

## Overview
This project demonstrates a practical ethical hacking assessment performed in a controlled virtual environment using Kali Linux and a Windows 7 target machine. The objective was to understand how vulnerabilities in legacy operating systems can be identified, analyzed, and exploited using industry-standard penetration testing tools.

The project was completed as part of the **Cyber Security and Ethical Hacking** training program at **VEDIC – VDIA Engineering Design Innovation Center**.

## Objectives
- Understand the penetration testing lifecycle.
- Perform network reconnaissance and host discovery.
- Identify vulnerabilities in a Windows 7 system.
- Analyze and exploit the MS17-010 (EternalBlue) vulnerability.
- Gain hands-on experience with Metasploit Framework.
- Understand post-exploitation concepts and security implications.
- Learn the importance of patch management and system hardening.

## Methodology

### 1. Environment Setup
- Configured Windows 7 and Kali Linux virtual machines.
- Established network connectivity between both systems.

### 2. Reconnaissance
- Identified active hosts within the local network.
- Collected target system information.

### 3. Vulnerability Scanning
- Performed vulnerability assessment using Nmap.
- Identified the MS17-010 (EternalBlue) vulnerability.

### 4. Exploitation
- Utilized the appropriate Metasploit module.
- Established a Meterpreter session on the target machine.

### 5. Post-Exploitation
- Performed system enumeration.
- Analyzed user and system information.
- Assessed the security impact of successful exploitation.
