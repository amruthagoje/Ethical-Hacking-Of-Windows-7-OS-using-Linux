# Ethical-Hacking-Of-Windows-7-OS-using-Linux-
Performed ethical hacking on a Windows 7 virtual machine using Kali Linux in a VirtualBox-based lab environment. Conducted network scanning, vulnerability assessment, exploitation of the MS17-010 (EternalBlue) vulnerability, and post-exploitation analysis using Nmap, Metasploit, and Meterpreter while following ethical cybersecurity practices.
